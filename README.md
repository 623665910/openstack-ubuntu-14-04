OpenStack
=========

《OpenStack开源云王者归来》一书所附代码及资料。本书采用OpenStack Grizzly版本。系统使用Ubuntu-14.04。

    - 由于在安装时，发现使用Desktop源是位于国内，速度较快，值得推荐。
    - 如果安装的是ubuntu-14.04-server amd64，那么需要把us.archive.ubuntu.com改为cn.archive.ubuntu.com。
